## [0.1.0] - 2026-02-04

### Added
- MCP server implementation with JSON-RPC 2.0 over stdio
- 14 built-in tools for ServerPod project analysis
- Project root detection for ServerPod v3 monorepos
- YAML parser for .spy.yaml model definitions
- Method signature parser for endpoints
- Service locator for centralized project access
- Resources and prompts support (MCP protocol)
- Comprehensive logging with colored output
- Integration tests against Pilly project
